## prepare

- 進到 r2 的頁面，右邊 "管理 R2 API 權杖"，獲得 R2_ACCESS_KEY_ID 和 R2_SECRET_ACCESS_KEY
- env 需要透過 api 傳入其他檔案

## install

```
bun add @aws-sdk/client-s3
```
